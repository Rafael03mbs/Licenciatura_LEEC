# Projeto: Simulação de um Conveyor de Frutas em Python

Este projeto é uma simulação orientada a objetos de um sistema industrial de um tapete rolante para a inspeção e separação de frutas. O objetivo é simular a passagem de diferentes tipos de fruta, a sua deteção por um sensor e a sua separação por um atuador.

### Funcionalidades Principais
*   Modelação de diferentes tipos de fruta como objetos com propriedades distintas (tamanho, cor).
*   Simulação do movimento contínuo do tapete rolante.
*   Implementação de um "sensor" virtual que identifica as propriedades da fruta.
*   Implementação de um "atuador" virtual que desvia a fruta para o cesto correto com base nos dados do sensor.

### Tecnologias Utilizadas
*   **Linguagem:** Python
*   **Padrão de Desenho:** Programação Orientada a Objetos
